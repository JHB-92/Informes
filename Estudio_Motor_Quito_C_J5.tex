Siendo:
\begin{itemize}
    \item \textbf{P1}: Potencia el칠ctrica de entrada
    \item \textbf{P2}: Potencia Mec치nica calculada
    \item \textbf{\eta}: El rendimiento calculado
    \item \textbf{s}: el deslizamiento en tanto por ciento
    \item \textbf{$M_{calculado}$}: El momento calculado

La potencia $P2$ se obtiene de multiplicar la velocidad angular por el momento calculado. Lo primero que se hace es calcular la velocidad angular con la siguiente ecuaci칩n:
\begin{equation}
{\omega = \frac{f \cdot 2 \phi }{N \cdot \left(1 - \frac{s}{100}\right)}}
\label{eqn:velocidad_angular}
\end{equation}

donde:
\begin{itemize}
    \item \textbf{\omega}: velocidad angular
    \item \textbf{f}: frecuencia en Herzios. Para este c치culo se emplean 75 Herzios
    \item \textbf{N}: n칰mero de polos del motor
    \item \textbf{s}: el deslizamiento
\end{itemize}
Por otro lado, la potencia mec치nica medida se calcula como el producto de la velocidad angular por el momento medido y, por 칰ltimo, el rendimiento se calcula dividiendo la potencia mec치nica medida por la potencia el칠trica.
